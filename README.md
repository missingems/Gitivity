# Gitivity 🟩 

Gitivity is a lightweight, native macOS menu bar application that keeps your GitHub contribution streak front and center. Built entirely in modern Swift and SwiftUI, it polls your GitHub GraphQL data in the background and renders your last 5 days of activity using the classic GitHub green squares. 

### Features
* **Live Tracking:** Automatically fetches your contribution history via the GitHub GraphQL API.
* **Secure Storage:** Your Personal Access Token (PAT) is safely encrypted in the native macOS Keychain.
* **Unobtrusive:** Runs as a pure UIElement (Agent) app, meaning it stays out of your Dock and application switcher.
